Backup

Export critical CA and Step-CA UI data for manual restore

Backup bundle

The archive contains a manifest with SHA-256 checksums, a PostgreSQL dump, and archives of the Docker volumes accessible to the Step-CA UI container.

PostgreSQL postgres-stepui.sql
Step-CA data step-ca-data.tgz from /home/step
Step-UI data step-ui-data.tgz, step-ui-certs.tgz, step-ui-uploads.tgz
Manifest manifest.json with versions, sizes and checksums
Important notes
  • The backup contains CA private keys. Store the archive as a secret.
  • Restore from the UI is intentionally not implemented: restoring could overwrite CA keys and the database.
  • Проверенная ручная процедура описана в BACKUP_RESTORE.md.
  • Before updates you can also create a CLI backup: make backup.
