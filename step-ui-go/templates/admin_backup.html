Бэкап

Экспорт критичных данных CA и Step-CA UI для ручного восстановления

Backup bundle

Архив содержит manifest с SHA-256 checksums, дамп PostgreSQL и архивы Docker volumes, доступные контейнеру Step-CA UI.

PostgreSQLpostgres-stepui.sql
Step-CA datastep-ca-data.tgz из /home/step
Step-UI datastep-ui-data.tgz, step-ui-certs.tgz, step-ui-uploads.tgz
Manifestmanifest.json с версиями, размерами и checksums
Важные условия
  • Backup содержит приватные ключи CA. Храните архив как секрет.
  • Restore из UI намеренно не реализован: восстановление может перезаписать CA keys и базу.
  • Проверенная ручная процедура описана в BACKUP_RESTORE.md.
  • Перед обновлениями можно также использовать CLI: sudo ./install.sh --mode backup --lang ru.
