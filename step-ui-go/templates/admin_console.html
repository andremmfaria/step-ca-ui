Консоль сервера

Прямой доступ к выполнению команд внутри контейнера

Требуется двухфакторная аутентификация

Консоль даёт полный shell внутри контейнера — потенциально разрушительная операция. Для активации настройте 2FA в профиле администратора.

Настроить 2FA ← Назад к панели
Что будет после 2FA
  • Интерактивный shell внутри контейнера step-ui
  • История команд и автодополнение
  • Предустановленные команды: restart step-ca, reload, backup
  • Все действия логируются в журнал безопасности
  • Автоматическая блокировка опасных команд (rm -rf /, dd, mkfs)
