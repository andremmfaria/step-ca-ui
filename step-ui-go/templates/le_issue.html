Выпустить LE сертификат

Новый сертификат

Домен должен быть публично доступен
Для уведомлений Let's Encrypt
HTTP-01: Let's Encrypt обратится на http://example.com/.well-known/acme-challenge/. Порт 80 должен быть открыт и доступен из интернета.
DNS-01 Cloudflare: Автоматически создаёт TXT запись через Cloudflare API. Настрой API Token в Настройках LE.
DNS-01 Route53: Автоматически создаёт TXT запись через AWS Route53. Настрой ключи в Настройках LE.

Как это работает

Let's Encrypt выпускает бесплатные SSL сертификаты для публичных доменов сроком на 90 дней.

HTTP-01

Подходит если сервер доступен из интернета на порту 80.

DNS-01

Подходит для любых доменов. Требует доступ к DNS через API.

Авто-обновление проверяет сертификаты каждые 24 часа и обновляет за 30 дней до истечения.
